init - инициализация(создание)
log - логирование(история)
add - добавить
commit - сохранение
Алгоритм 
1.   Проверить настройки - config
2.   Выбрать папку для отправки
3.   Создать/проверить что в папке есть .git
4.   Добавить файлы
5.   Сохранить
6.   Указать репозиторий куда нужно файлы отправить
7.   Отправить файлы
